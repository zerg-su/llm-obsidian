# С чего начать в llm-obsidian

Этот вольт — ваша компаундящая база знаний. Persistent second brain на стыке Claude и Obsidian.

Каждый ингестнутый источник обрабатывается в 8-15 cross-referenced wiki-страниц. Каждый вопрос синтезируется из всего что было прочитано ранее. Знания накапливаются как сложный процент.

---

## Три шага quick start

### 1. Положите источник

Положите любой документ в папку `.raw/`:
- PDF, markdown-файлы, транскрипты, статьи
- Либо передайте URL и попросите Claude забрать

### 2. Ingest

В любой Claude Code сессии:

```
ingest <имя-файла>
```

Claude читает источник, создаёт 8-15 wiki-страниц под `wiki/`, cross-reference'ит всё, и обновляет `wiki/index.md`, `wiki/log.md`, `wiki/hot.md`.

### 3. Задавайте вопросы

```
что ты знаешь про <тема>?
```

Claude читает hot cache, сканирует index, drill в релевантные страницы, синтезирует ответ с цитатами на конкретные wiki-страницы (не из training-knowledge).

---

## Как работает Hot Cache

`wiki/hot.md` это ~500-словное саммари свежего контекста вольта. Грузится автоматически в начало каждой сессии через SessionStart-хук.

Не нужно пересказывать что было. Claude стартует каждую сессию зная над чем вы работаете.

Обновить вручную в любой момент: `update hot cache`.

---

## Walkthrough первого ингеста

1. Создать файл в `.raw/` — скопировать транскрипт, вставить статью, сохранить PDF
2. Открыть Claude Code в папке вольта
3. Ввести: `ingest <имя-файла>`
4. Наблюдать как растёт вики — Claude репортит созданные страницы
5. Открыть `wiki/index.md` — новые страницы в каталоге
6. Открыть Graph View в Obsidian — появился новый cluster связанных нод

После 3-5 ингестов граф начинает выглядеть как настоящая сеть знаний. Cross-reference'ы возникают автоматически.

## Ключевые команды

| Что говорю | Что делает Claude |
|---|---|
| `ingest <файл>` | Создаёт 8-15 wiki-страниц из источника |
| `что ты знаешь про X?` | Запрашивает вики, цитирует страницы |
| `/save` | Файлит текущий разговор как wiki-заметку |
| `/autoresearch <тема>` | Ищет в вебе, ингестит результаты автономно |
| `lint the wiki` | Health check — находит orphans, пробелы, stale links |
| `update hot cache` | Обновляет саммари контекста сессии |
